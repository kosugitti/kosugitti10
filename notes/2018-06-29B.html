標準正規分布のテイルを塗りつぶす

2018-06-29

標準正規分布のテイルを塗りつぶす。

標準偏差+2以上のテイル

library(ggplot2)
library(gridExtra)
limitRange <- function(fun, min, max) {
  function(x) {
    y <- fun(x)
    y[x < min  |  x > max] <- NA
    return(y)
  }
}
# p(x>2)
p <- ggplot(data.frame(x=c(-4, 4)), aes(x=x))
p + stat_function(fun = dnorm) +
  stat_function(fun = limitRange(dnorm, 2, 4),
                geom="area", fill="blue", alpha=0.2)
